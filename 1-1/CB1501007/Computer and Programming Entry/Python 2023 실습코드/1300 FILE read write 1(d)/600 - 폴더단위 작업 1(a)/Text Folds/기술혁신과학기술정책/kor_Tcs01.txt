델파이법은 익명성 통계적 분포 제시 추론의 피드백과 반복 설문이라는 세 가지 특성을 갖는다 전통적인 델파이법은 라운드로 진행되는데.

플러그인 기반의 인체모형 시각화 은 웹상에서 가상현실을 구현하기 위한 언어이다.
은 인터넷상에서 가상환경을 구축할 수 있는 국제적 표준으로 정해졌으며 이후에 혹은 가 출시되었다.
의 장점은 웹브라우저에서 읽히는 과 비슷한 형식의 텍스트 파일이라는 점이다.
그렇기에 플랫폼에 영향을 받지 않고로 변환하는 과정도 필요 없기 때문에 렌더링 시에 그래픽카드에 의한 이미지 가속도 필요로 하지 않는다 또한 파일은 웹브 하우저에 플러그인만 설치하면 매우 쉽게로 시각화할 수 있다.
하지만 플러그인에 의해 구 동 되기에 시각화 시스템에 원하는 기능을 추가하기가 극히 어려운 문제점이 있었다.
기반의 인체모형 시각화 파일은 기반의 표준 파일로의 차세대 파일이다.
파일을 시각화하기 위하여 라이브러리를 사용하였다는에서 진행 중인 과 컨텐트를 자바로 완벽하게 지원하는 툴킷 개발을 위한 프로젝트이다 라이브러리는 거를 기반으로 하기 때문에 다양한 플랫폼으로 이식 성이 좋은 특징이 있다.
또한 이 시스템을 통해서 스펙을 검증하는 테스트 베드로서의 역할을 수행하고자 하였다 하지만 가 워낙 대중화되지 못한 라이브러리였기 때문에 원하는 기능을 구현하기 위해 관련 자료가 충분치 못했기 때문에 형태의 개발 진행 속도가 너무 느려 개발 계획을 바꿔야만 했다.
그림 기반의 인체모형 시각화 프로토타입 시스템 애플릿 기반의 인체모형 시각화는 자바 플랫폼 상에 기반의 개발 이다는 이 나 등 어디에서도 동작이 가능하며 현재 구동되는 인체모형 시각화 버전은 상에서 동작하도록 개발되었다를 이용하여 개발하기 위해서 머리 모형은 가장 범용적인 파일인 파 일 및 파일로 변환하여 사용하였다.
이 두 파일 역시 파일 등과 같 이 텍스트 파일이지만 앞의 두 파일은 태그 정보에 의해 구조화된 형식을 가지지만 파일은 매우 단순하게 데이터를 열거한 형태의 단순구조를 가진다는 강력한 개발 로 선으로 개발하여 웹에 쉽게 부착할 수 있는 장점이 있다.
또한 을 활용하여의 인터페이스를 원하는 대로 수정이 가능하기 때문에 개발도 용이한 장점이 있다.
다만 기반으로 구동 시에는 많은 메모리 공간을 요구하며에 서 동작하기에 속도가 느린 단점이 있다.
우측 상단의 시각화 화면은 용어 계층 화면에서 선택된 인체모형들을 가시적으로 보여주는 화면으로 마우스와 기능 버튼을 통해서 사용자는 회전 이동 확대 등을 수행하여 다양한 방 향 및 크기로 인체모형을 살펴볼 수 있도록 하였다.
우측 하단의 컨트롤 화면은 선택된 인체모형의 정보를 표시하거나 추가적인 기능을 수행하는 화면이다.
체크박스는 활성화 상태 비활성화 상태 부분 활성화 상태의 가지 상태를 갖는다 활성화 상태는 화면에 해당 모형이 보이는 상태 비활성화 상태는 보이지 않는 상태이며 부분 활성화 상태는 상위계층의 모형에서만 나타나는 상태로 하위 인체모형 중에서 일부 모형만 화면에 보이고 있는 상태를 나타낸다.
좌측의 용어 체계 트리구조에서는 상위 계층의 이름을 마우스로 클릭하여 선택하면 하위 계층의 인체모형들이 동시에 선택되도록 하였다.
또한 추가적인 선택 방법으로 우측 하단의 컨트롤 화면에서 검색을 통해서도 선택이 가능해.
검색 기능 인체모형 시각화에 사용된 해부학 용어 체계에 따라 계층구조로 정리되어 있는 인체모형 내 역은 해부학에 대한 지식이 있지 않다면 쉽게 이해되지 않는다 따라서 이름만 알고 있는 인체 모형에 대해서는 우측 하단에 위치한 컨트롤 화면에서 검색을 통해서 인체모형을 선택할 수 있도록 하였다.
검색한 문자열을 포함하고 있는 인체모형이 다수 존재한다면 검색 버튼을 반복해서 클릭함으로써 다음에 위치하는 문자열 매칭된 인체모형을 차례로 선택할 수 있다.
강조 효과 인체모형을 선택하더라도 여러 개의 모형이 동시에 보이는 화면에서 현재 선택된 모형을 사용자에게 쉽게 알 수 있도록 하기 위해서는 화면상에 특수한 효과를 주어야만 한다.
따라서 이를 위해서 선택된 모형에 대해서는 강조 효과를 줄 수 있도록 하였다.
우측 하단에 위치한 컨트롤 화면에서 강조 효과 체크박스에 체크를 하면 현재 선택된 인체모형에 황색 계열의 빛 효과를 줌으로써 시각적으로 두드러지게 보이도록 하는 강조 효과를 주었다.
강조 효과 체크박스가 체크가 된 이후에 다른 인체모형을 선택하면 강조 효과가 새로 선택된 인체모형으로 옮겨지게 된다.
즉 강조 효과는 현재 선택된 인체모형에 대해서만 작동하도록 구현되었다.
앞서 설명했듯이 선택은 상위계층에 대해서도 가능하므로 상위계층을 선택하는 경우에는 하위에 포함된 모든 인체모형에 대해서 강조 효과가 부여된다.
강조 효과를 취소하고 싶으면 다시 강조 효과 체크박스에 체크를 하여 비활성화 시키도록 한.
투명화 효과 다수의 인체모형이 동시에 보이게 되면 이리저리 돌려보고 확대해 보아도 뒤에 감춰 지거 나 안에 숨겨진 인체모형은 보이지 않는다 따라서 이러한 인체모형을 보기 위해서 투명화 효과를 추가하였다.
투명화 효과는 강조 효과와는 다르게 일시적인 효과가 아니라 지속적인 효과이다.
선택한 인체모형에 투명화 효과를 부여하고 다른 인체모형을 선택했다가 다시 기존에 투명화 효과를 부여했던 인체모형을 선택하게 되면 여전히 투명화 효과를 유지하고 있음을 알 수 있을 것이다.
투명화 효과도 역시 활성화 상태 비활성화 상태 부분 활성화 상태의 세 가지 상태를 갖는다 활성화 상태는 선택된 인체모형이 투명화가 된 상태이며 비활성화 상태는 불투명화 상태인 경우이다.
부분 활성화 상태는 상위계층에 대해서만 나타나는 상태로 하위에 포함된 인체모형 중에서 일부 모형이 투명화 효과가 부여되고 있음을 표시한다 즉 상위계층에 대해서 투명화 효과가 활성화 상태인 경우에는 하위의 모든 인체모형이 투명화 효과가 부여되고 있음을 의미한다.
회전 기능 차원으로 구현된 인체모형을 여러 방향에서 바라볼 수 있도록 회전 기능도 기본적으로 존 재한 다 회전은 마우스 좌측 버튼의 드래그 앤 드롭 혹은 화면 좌측 하단에 위치한 기능 버튼의 대각선에 위치한 개 버튼을 이용하여 회전을 할 수 있다.
마우스로 회전하는 경우에는.
우스를 화면상에서 클릭하고 원하는 방향으로 마우스를 끌다가 놓으면 쉽게 회전이 가능하다 하지만 기능 버튼을 이용하는 경우에는 대각선의 네 방향으로만 회전이 가능하므로 마우 그 조작보다는 원하는 방향에서 바라보도록 회전하기는 쉽지 않다.
그렇더라도 미세한 조작이 필요하다면 마우스보다는 기능 버튼의 조합으로 회전하는 것이 유리한 경우도 있을 수 있다.
인체모형의 세부 형태를 파악하기 위해서 확대 및 축소 기능은 매우 유용한 기능이다.
이동 기능 인체모형은 서로 다른 위치에 다른 크기로 존재하고 있기 때문에 원하는 인체모형을 화면의 중심에 두기 위해서는 화면을 이동하는 기능이 필요하다 화면을 이동하여 확대 및 축소 기 능을 사용하여 원하는 방향 원하는 크기로 인체모형의 세부 형태를 파악할 수 있다.
화면의 이 동을 위해서는 마우스 오른쪽 버튼을 누르고 드래그 앤 드롭을 수행하거나 좌측 하단의 기능 버튼 중 상하좌우의 삼각형 모향의 버튼을 클릭하여 가능하다 마우스를 이용하여 이동할 때 방향을 사용자가 원하는 아무 방향으로 설정 가능하기에 사용법이 쉬운 장점이 있다.
기능 버튼은 상하좌우 네 방향으로만 이동이 가능하므로 미세한 이동이 필요할 때 사용하도록 한다.
추가 기능 소개 인체모형 시각화에는 다른 인체 시각화 프로그램과는 차별되는 다음의 기능을 제공한다 위키피디아 연결 기능 선택된 인체모형의 전문적인 정보를 사용자에게 보여주기 위해 위키피디아를 선택하여.
위키피디아 백과사전 위키피디아는 웹 기반으로 전 세계의 사용자들의 협업으로 구축되고 있는 다국 어 백과사전이다.
영어 위키피디아의 경우 년 월 현재 개 이상의 엔트리를 포 함하고 있으며 한국어 위키피디아의 경우에는 현재 개 이상의 엔트리를 포함하고 있다.
세계에서 가장 영향력 있는 웹사이트로도 평가받고 있지만 많은 사람들이 수정하는 자료이니 만큼 자료의 신뢰성에는 논란의 소지가 있다.
위키피디아와의 연결 인체 모형 시각화에는 좌측의 해부학 용어 체계 혹은 용어 체계의 용어로부터 필요한 경우 사용자가 위키피디아를 즉시 참조할 수 있는 기능을 추가하였다.
절단면 기능을 시작하면 화면의 좌상단에는 작은 크기의 모형을 보여주며 붉은색의 평 면으로써 절단면의 위치를 표시해준다 화면 중앙에 위치한 그림은 해당 절단면을 기준으로 머리 위에서부터 아래로 내려다본 그림이다.
마우스 휠을 앞뒤로 돌리면서 절단면의 위치를 위아래로 이동할 수 있다.
절단면 기능 수행 중에는 마우스의 드래그 앤 드롭을 통한 회전 및 이동은 동작하지 않는다 또한 좌하단의 기능 버튼은 좌상단의 작은 크기의 모형을 회전 이동 확대하는 작업을 수행한다 그림 절단면 기능의 예 화면 출력 기능 사용자가 화면을 조작하다가 원하는 구도의 이미지를 찾으면 캡처 버튼을 통해서 화면을 저장할 수 있도록 하는 기능을 추가하였다.
버튼을 클릭한 후에 파일 이름을 입력하고 저장을 누르면 형태의 파일로 저장이 된다.
저장된 파일은 교육 등의 목적으로 활용이 가능하며 시스템에는 인체모형 게시판을 추가하여 다른 사람들과의 정보교환을 위한 목적으로 활용하기 위한 환경을 마련하였다.
연속 절단면 영상의 경우는 거의 간격으로 절단하여 파일에 해상도인으로서 모두 장의 영상으로 이루어졌고 전체 데이터의 크기는 약에 달한다 그림 연속 절단면 영상 제작 과정 인체 머리 부위의 데이터를 통해 제공받은 인체 머리 부위의 데이터는 간격으로 촬영된 머리의 연속 절단면 영상과 연속 절단면 영상을 바탕으로 제작된 머리의 구역화 영상이다.
머리의 연속 절단면 영상의 파일 포맷인으로 해상도의 고해상도 이미지로써 개의 파일 총용량을이다.
구역화 영상은 연속 절단면 영상에 해부구조물의 테두리를 그린 다음에 각 해부구조 물의 테두리 속에 구조물 별로 구분하기 위한 임의의 색상의 조합을 채워서 구역 화 영상 해상도 을 만든 것이다.
